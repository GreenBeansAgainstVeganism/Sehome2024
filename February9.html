February 2nd Agenda!

Set up your github account:

  1. Go to github.com
  2. If you have an account already, sign in. If not, click sign up.
    • I strongly reccommend using a personal email to create your account so that you will continue to have access to it after leaving school.

Install Github Desktop:

  1. Visit desktop.github.com
  2. Click Download for Windows (64bit)
  3. Locate and run the installer
